Getting Started

  1. Go to option Upload Raw Data from the main menu.
  2. Use the new form to upload your json file.
  3. Once the file is uploaded, its content is divided and stored as a number of entities. Each entity respresents one object from the uploaded json file. Now, click the option Analyse Data from the main menu.
  4. Fill in the new form and configure your classification: it will identify the groups among all the entities in the database and then sort them accordingly. Parameters of the classification are:
    • similarity threshold (the percentage of attributes (0 - 100) which can differ among the entities within a group)
    • maximum number of entities to be involved in the classification
      Note: If this is not specified, the classification will try to classify all entities which do not belong to any group.
    • description
  5. Once the classification is finished, you will be redirected to the results page which contains:
    • information about predefined classification parameters
    • list of groups which where identified and their attributes
    • chart showing identified groups
  6. With these groups, you can now analyse the dependancies between certain attribute values among the entities of the same group. Click on Plot Data option from the main menu.
  7. Choose New Presentation to create a new chart to illustrate the dependencies.
  8. Once you fill in the new form and click on Create Presentation, a chart will be shown. You can access it later by visiting Plot Data since the list of all charts is kept there.
